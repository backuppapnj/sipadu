06 — Keamanan

Arsitektur Keamanan

Implementasi keamanan sesuai Perpres 95/2018, BSSN 4/2021, dan SNI ISO 27001

🔐

Enkripsi NIK

AES-256-CBC via Laravel Crypt. NIK dienkripsi sebelum disimpan ke database sesuai UU PDP 27/2022.

at-rest encryption
📎

File Security

Lampiran dienkripsi at-rest, disimpan di private disk, dengan SHA-256 checksum per file.

max 10MB · 5 files
🔑

Session Auth

Laravel Sanctum session-based. Cookie HttpOnly + Secure + SameSite. CSRF otomatis via Inertia.

sanctum session
🚫

Account Lockout

Otomatis setelah 5 gagal login, durasi 30 menit. Semua percobaan dicatat ke audit log.

5 attempts · 30 min
📝

Immutable Audit

Tabel audit_logs tidak dapat diubah atau dihapus. Mencatat: siapa, kapan, dari mana, apa, sebelum/sesudah.

11 fields · immutable
🗑️

Soft Delete

Seluruh model utama menggunakan soft delete. Data tidak pernah dihapus permanen — SPBE compliance.

no hard deletes
🔤

Password Policy

Min 8 karakter, huruf kapital, angka, simbol. Peringatan kedaluwarsa 90 hari (konfigurasi).

strong password
🏷️

Data Classification

Setiap tabel dan record ditandai: PUBLIK / INTERNAL / RAHASIA. Akses difilter berdasarkan role.

3 levels
🆔

Request Tracing

Setiap response menyertakan header X-Request-ID (UUID) untuk traceability sesuai BSSN.

uuid per request
